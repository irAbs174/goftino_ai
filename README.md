Install :
1. Rename app/core/simple_conf.py to app/core/conf.py with execute:
``` bash
mv app/core/simple_conf.py app/core/conf.py
```
2. setup your app/core/conf.py configure(Api)
3. execute and enjoy :
``` bash
chmod 755 app/deploy.sh && ./deploy.sh
```
